# Why Blog

- 对于自己的学习经历做一个记录，也方便督促自己。
- 把自己对于知识的理解尽可能明白的总结出来，方便其他人的学习。
- 最近一段时间主要是计算机图形学基础的学习，未来涉及有高级渲染、游戏开发、游戏引擎开发等方面。
